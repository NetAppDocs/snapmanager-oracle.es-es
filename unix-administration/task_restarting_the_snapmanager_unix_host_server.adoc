= Reiniciar el servidor host UNIX de SnapManager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede reiniciar el servidor en un host UNIX utilizando la CLI.

. Para reiniciar el servidor, introduzca el siguiente comando:
`smo_server restart`
